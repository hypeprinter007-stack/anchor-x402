# anchor-x402

> English / 英語版: [llms.txt](https://anchor-x402.com/llms.txt)

> AIエージェント向けのx402従量課金サービス16種（Base + Solanaメインネット対応）と、chat.anchor-x402.comで提供されるホスト型チャットボット。コール単位のUSDC決済で、APIキー・アカウント登録・サブスクリプション、いずれも不要です。

anchor-x402は、AWS Lambda + FastAPI上で動作する16種のステートレスサービスを公開しています。各エンドポイントは `extensions.bazaar` を含むx402 v2のPaymentRequiredチャレンジを返却し、クライアントはBaseまたはSolanaの任意のウォレットから、EIP-3009 transferWithAuthorizationを用いて1コールあたり $0.001〜$1.77 USDCを支払います。決済はCoinbase CDPファシリテーター経由。本番稼働中：https://api.anchor-x402.com

## ホスト型エージェントチャットボット

[chat.anchor-x402.com](https://chat.anchor-x402.com) — 16種のサービスをユーザーに代わって実行するホスト型Claudeエージェント。自身のエージェントを持たないユーザー向けの推奨経路です。ユーザーはCoinbase Smart Wallet（パスキー対応、アプリ不要）または任意のブラウザ拡張ウォレット（MetaMask、Rabby、Frame等）を接続し、コール単位で支払います。チャット自体は無料、有料ツールコール毎にユーザーの明示的な承認が必要で、EIP-3009 USDC認可署名を1件発行します。セッション支出上限はユーザー側で設定し、クライアント側で強制されます。

非開発者ユーザーにとっての推奨経路であり、16サービスの公式デモ面でもあります。各サービスは `/demo <name>` で無料デモ出力が利用可能です（支払不要）。

## エージェントからのHTTP API利用方法

[anchor-x402-mcp（npm）](https://www.npmjs.com/package/anchor-x402-mcp) — 9種の汎用プリミティブサービスをツールとして公開するMCPサーバー。Claude Desktop、Claude Code、Codex CLI（OpenAI）、ChatGPT Desktop、Cursor、OpenAI Agents SDK、その他MCP互換クライアントから利用可能。

```json
{
  "mcpServers": {
    "anchor-x402": {
      "command": "npx",
      "args": ["-y", "anchor-x402-mcp"],
      "env": { "ANCHOR_WALLET_PRIVATE_KEY": "0xYOUR_BASE_WALLET_KEY" }
    }
  }
}
```

MCPラッパーは、@x402/fetch + @x402/evm経由で各コールをエージェントのBaseウォレットから自動決済します。プリペイ不要、APIキー不要、各エンドポイント実装内の制約以外のレート制限なし。

または任意のx402 v2クライアントSDK（[@x402/fetch（npm）](https://www.npmjs.com/package/@x402/fetch)、Python x402 SDK、Rust）でHTTP APIを直接コール可能です。決済フローは共通：402 → 署名 → 再送。

## オンチェーン検証可能性

`/v1/anchor`、`/v1/attest`、`/v1/oracle`、および `/v1/investigate` レポートは、32バイトハッシュを **Base ＋ Solanaメインネットの両方** に書き込みます。オンチェーンに記録されたバイト列はサービスから独立しており、anchor-x402への問い合わせ不要で、誰でもチェーンを読むだけでレシートを検証できます。改ざんにはSHA-256の破壊または2つのL1の同時リオーグが必要です。

ライブメインネット・ハッシュを用いた実例：https://anchor-x402.com/trust/on-chain-verifiability
